# 3D Print Kits

This folder contains exported `.3mf` print kits for the physical prototype.

Examples:

- `level0-ui-cover-plate.3mf`
- `photo-booth-tray.3mf`
- `servo-holders.3mf`
- `level2-motor-hub-receiver.3mf`
- `level2-socket-hub-arms.3mf`
- `kw11-trigger-and-prism.3mf`
- `structure-brackets_long.3mf`
- `structure-brackets_short.3mf`

## Final Prototype Note

The final bucket compartments were made from manually cut styrene sheets. Any bucket-divider print kits from earlier iterations should be treated as design history rather than the final physical storage solution.
