# Manufacturing Files

This folder contains fabrication references for the physical R3Bin prototype.

## Files

- `PVC_Layout_R3.DXF`: cutting layout for PVC panels.
- `Cortes PVC 5mm.pdf`: cutting reference/specification document.

## Materials

- 5 mm foamed PVC for the main body.
- Acrylic rear panel for process visibility.
- PETG/PLA printed mechanisms and structural parts.
- 30 pt styrene sheets manually cut for bucket divisions.
